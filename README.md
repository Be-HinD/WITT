<h2 id="section1">👀 1. 서비스 소개 </h2>
WITT는 분리수거에 대한 정보를 잘 모르는 사람을 대상으로 하는 서비스입니다.
<br>
이미지 분류모델을 통해 분리수거에 대한 각종 퀴즈를 풀며 얻은 재료를 얻을 수 있습니다.<br>
사용자는 재료를 통해 자신만의 캐릭터를 성장시켜나가는 시뮬레이션 서비스입니다.
